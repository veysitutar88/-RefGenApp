# UI Notes

## Layout
- Left sidebar navigation
- Center workspace (lists, forms, detail)
- Right inspector showing JSON + quick actions

## Key components
- `Layout` — top-level shell + sidebar + inspector
- `Inspector` — JSON viewer and action buttons

## Routes
- `/outputs` — Outputs feed
- `/characters` — Character library
- `/shots` — Shot library
- `/run` — Run screen
- `/outputs/:outputId` — Output detail
- `/exports` — Export manifest viewer
- `/settings` — Settings stub
